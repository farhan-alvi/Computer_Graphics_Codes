# Computer_Graphics_Codes
Computer Graphics projects implemented using OpenGL in C++. Includes basic rendering, transformations and 2D/3D shapes.
## Technologies Used
- C++
- OpenGL
- GLUT
- Code::Blocks

## Features
- 2D and 3D object drawing
- Transformations (rotate, translate, scale)
- Basic animations

## How to Run
1. Open the project in Code::Blocks
2. Make sure OpenGL and GLUT are properly set up
3. Build and run the project

## Purpose
This project was developed as part of my Computer Graphics course. It focuses on understanding the fundamentals of graphics programming using OpenGL.
